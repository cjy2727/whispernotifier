# WhisperNotifier

## 소개
WhisperNotifier는 월드 오브 워크래프트(WoW)에서 귓말을 수신할 때 화면에 알림 메시지와 사운드를 표시해주는 애드온입니다. 중요한 귓말을 놓치지 않도록 도와줍니다.

## 주요 기능
- 귓말 수신 시 화면에 알림 메시지 표시
- 사운드 알림 지원
- 알림 메시지, 글자 크기, 위치 등 사용자 설정 가능
- 테스트 알림 기능 제공
- 음소거(Mute) 기능
- 볼륨 조절 기능

## 설치 방법
1. 이 저장소를 다운로드하거나 압축을 풉니다.
2. `WhisperNotifier` 폴더를 다음 경로에 복사합니다:
	- `World of Warcraft/_retail_/Interface/AddOns/`
3. 게임을 실행한 후 애드온 목록에서 WhisperNotifier를 활성화하세요.

## 사용법
애드온 설정에서 알림 메시지, 글자 크기, 위치 등을 원하는 대로 조정할 수 있습니다. 테스트 버튼을 눌러 알림을 미리 확인할 수 있습니다.

## 파일 설명
- `whispernotifier.lua`: 애드온의 주요 기능을 담당하는 Lua 스크립트
- `whispernotifier_config.lua`: 애드온의 설정을 담당하는 Lua 스크립트
- `whispernotifier.toc`: 애드온의 메타 정보 및 파일 목록
- `whispernotifier.png`: 애드온 아이콘 이미지
